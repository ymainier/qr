# QR Code Generator and Scanner

This application provides QR code generation and scanning functionality built with React and Vite. It features:

- QR Code Generator at `/qr/generate`
- QR Code Scanner at `/qr/scan`
- Deployed at [https://ymainier.github.io/qr/](https://ymainier.github.io/qr/)
- Uses [wouter](https://github.com/molefrog/wouter) as a lightweight frontend router, react, vite and typescript.
